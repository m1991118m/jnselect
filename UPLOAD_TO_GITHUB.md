# JN Select GitHub Pages 上傳步驟

1. 到 https://github.com/new 建立新 repository。
2. Repository name 建議填：`jn-select-showcase`
3. Public / Private 都可以；若要免費 GitHub Pages，建議選 Public。
4. 建好後點 `uploading an existing file`。
5. 上傳這個資料夾內的所有檔案：
   - `index.html`
   - `style.css`
   - `script.js`
   - `products.js`
   - `.nojekyll`
   - `public/` 整個資料夾
6. 點 `Commit changes`。
7. 到 repository 的 `Settings` -> `Pages`。
8. Source 選 `Deploy from a branch`。
9. Branch 選 `main`，資料夾選 `/root`，按 `Save`。
10. 等 1 到 3 分鐘，GitHub 會產生公開網址。

公開網址格式通常會是：

`https://你的GitHub帳號.github.io/jn-select-showcase/`
